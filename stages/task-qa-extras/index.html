Instructions

Delete pathways tagged for deletion

The goal of this task is to removed pathways with expired deletion tags.

  • Follow the link to see a list of pathways tagged for deletion:

    • Pathways tagged for deletion
  • Check the last column for pathways that are ready to be deleted; look for the delete icon.
  • Simply click on the delete icon to delete these pathways
  • Note: Deletion can only be performed by sysop accounts

Jenkins Test Report

Test search

The goal of this task is to make sure that the search feature is working and the index is up-to-date.

  • Search for the newest pathway by title:
    • Copy the pathway title from the top of the New Pathways page listing, but make sure the pathway is public and not private.
    • Paste the title into the search field in the left sidebar
  • There may be many results, but be sure that the newest pathway is among them. If not, then send an email to the WP dev list describing the problem.

Admin curation logistics

  • Send a curation report to the development team summarizing your activities.
  • Update the Admin Curation Roster by adding a new start day and name to the top of the list. Take next name in the sequence established in prior weeks.
