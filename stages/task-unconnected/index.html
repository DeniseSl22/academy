WikiPathways: Connect Interactions to Nodes

Background

WikiPathways provides a space for pathway description. Like figure legends, this space should be used to give a concise explanation of purpose and important features of a pathway. Sometimes, however, a pathway author goes through all the trouble of submitting a great model but negelct to describe it. We need your help in providing high-quality descriptions for these pathways.
