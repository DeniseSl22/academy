Your Mission

Draw a metabolite.

  • To complete this challenge, click the "Launch Editor" button below. You will receive your challenge as a file named "WikiPathwaysAcademy-draw-metabolite.jnlp". Open it to see your control panel.
  • Perform the challenge tasks...
    • Select “Met“ from the Data Node palette in the Objects tab
    • Click on canvas to place the gene node
    • Double-click on node to open its properties
    • Search for “glucose”; Choose any of the entries that just say “glucose”; Click OK
    • (Optional) Resize nodes or update node labels
    • Done!
  • When you have completed the challenge, save your work as a GPML file under File > Save As.
  • Drag-n-drop the GPML file below to submit.
