WikiPathways: Add External References for Nodes

Background

For pathway models to be useful for data and network analysis, data nodes need to be annotated with database references. The "This pathway contains genes, proteins or metabolites with missing database annotations" tag at WikiPathways identifies pathways with unannotated data nodes. Our goal is to annotate as many data nodes on pathways as possible with proper database references.

Instructions

  • For the pathway below, follow the link to the Pathway at WikiPathways. You will need to be logged in at WikiPathways in order to access the editing controls.
  • The red "This pathway contains genes, proteins or metabolites with missing database annotations" tag will include a list of data nodes that are not annotated. In the pathway editor, annotate as many of these data nodes as possible.
  • For data nodes where it's not possible to find a match in any of the databases, leave it as-is and make sure to enter a note on the pathway discussion tab.
  • If all data nodes are annotated, save the pathway and remove the "This pathway contains genes, proteins or metabolites with missing database annotations" tag.
  • Click "Verify" below.
  • Click "Next" to load the next pathway.
