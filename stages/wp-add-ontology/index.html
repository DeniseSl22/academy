WikiPathways: Add Ontology Term

Background

WikiPathways uses ontologies to help categorize pathways to make them easier to find and sort. An ontology is simply a collection of terms arranged hierarchically, like a branching tree or folders with subfolders. We get our ontologies from another curated database dedicated to collecting and distributing biology-related ontologies, called BioPortal. For example, the have a Pathway Ontology. Whenever you create or edit a pathway you should take some time to annotate it with descriptive ontology terms.

Challenge

Add an ontology term to the Sandbox Pathway. You will have to be logged in at WikiPathways in order to access the Ontogoly Tags controls.


  • To complete this challenge, go to the Sandbox Pathway
  • Log in (link in upper-left corner) and return to the page
  • Scrol down to the Ontology Tags section
  • Type "tca" and choose the autocomplete answer, "citric acid cycle pathway", or search/browse for any term you'd like to add
  • (Optional) Click "i" for more information about this ontology term
  • Click the green checkmark to add the term to the pathways. Note: if the tag was already added, then remove it first, or add a different tag.
  • Done! Now, return to this page and click the "Verify" button below
