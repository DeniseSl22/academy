WikiPathways: Assess Recently Changed Pathways

Background

The goal of this task is to ensure that recent pathway edits are appropriate and that relevant tags are updated.

Instructions

  • For the pathway below, follow the link to the Pathway at WikiPathways. You will need to be logged in at WikiPathways in order to access the editing controls.
  • Perform the following tasks:

    • Review edit history

  • Scroll down to the History section and assess the comments for recent changes. Compare to views of prior revisions.
  • If the recent edits are inaccurate or degrade quality, revert the pathway to a revision prior to the latest set of edits.
  • If you reverted an edit (or are uncertain), start a discussion on the pathway and/or with the author directly, explaining your reasoning.

    • Community approval tags

  • If a changed pathway is tagged with a community approval tag, e.g., "Reactome Approved", collect a summary of pathway page links and change details to send to each community representative. If you are not already in contact the representative, simply include the summary of links and details in your wrap-up email to wikipathways-devel and we'll take it from there.

    • Add and update curation tags

  • Update faded tags if the recent edits are good by clicking on the click here link provided with "Curated collection" and "Featured pathway" tags.
  • Consider adding new tags, for example, if the following criteria are met:
    • Curated collection: All interactions must be complete, and most datanodes must be annotated. About 80% coverage of Gene Product datanodes is sufficient, given that some elements might not have mappings. Overall, the pathway should be useful for downstream data visualization and analysis.
    • Featured pathway: In addition to meeting Curated collection requirements, it must have a good description and one or more ontology tags. Non-typical pathways, including gene lists and abstract networks, should be excluded. Overall, a Featured pathway should have a consistent and informative annotated layout that contributes to the understanding of the biological process.
    • For both Featured and Curated collections, DO NOT include list-based pathways and other pathways that don't describe a biological process.
  • Remove tags that have been addressed and are now outdated. For example the "Missing description" tag, "Data nodes with missing database annotation" tag and "Incomplete interactions" tag.
  • Depending on which tags were updated, added or removed for each pathway, click the corresponding "Verify" buttons below. For example, if a description was added and the "Curated collection" tag was added, click the "Verify description modified" and "Verify Curated tag added".
  • Click "Next" to load the next pathway.
Note: The above criteria are used mainly for the archive of human and mouse pathways. For species other than human and mouse, these criteria are slightly more relaxed. Specifically, pathways don't need to have completed interactions, as long as most datanodes are annotated. Even without completed interactions, these pathways are useful for gene set analysis.
