WikiPathways: Assess Pathways Under Construction

Background

All newly created pathways at WikiPathways automatically get tagged with the "Under construction" tag. The purpose of this is to let the community know that the pathway may be incomplete. By default, a pathway is considered under construction for 30 days, after that the tag should be removed and if the pathway is not complete, other tags may be added to indicate that. The goal of this task is to only keep the "Under construction" tag on pathways that are still actively being edited.

Instructions

  • For the pathway below, follow the link to the Pathway at WikiPathways. You will need to be logged in at WikiPathways in order to access the editing controls.
  • Perform the following tasks:
  • Remove the "Under construction" tag from any pathway that either appears complete or is OVER 30 DAYS old and is no longer being actively edited. Then process as follows:
    • If the pathway looks like a tutorial or test pathway, or has no database annotations, no recent edits, no description AND unoriginal content, then add the "Proposed deletion" tag.
    • If the pathway has promise (e.g., some original content) but needs substantial work, then add the "Stub" tag, and add a note outlining necessary changes. Also consider starting a discussion under the "discussion" tab of the pathway.
    • If the pathway is reasonably complete, but could still use some work, then add the Needs work tag, and add a note outlining necessary changes. Also consider starting a discussion under the "discussion" tab of the pathway.
    • If the pathway is complete and meets the requirements for the Curated collection or Featured set, then add those tags at this time.
  • Depending on which tags were added or removed for each pathway, click the corresponding "Verify" buttons below.
  • Click "Next" to load the next pathway.
